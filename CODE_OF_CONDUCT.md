# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct.

The full text is published and maintained at the link above. By participating in this project (opening issues, sending pull requests, posting in Discussions, or any other interaction in this repository), you agree to abide by it.

In one line: **be kind, give people the benefit of the doubt, and don't make this project a hostile place.** That's the whole bar.

## Reporting

If you experience or witness behavior that violates the Contributor Covenant in this project's spaces, please email **michael@noisemeld.com**. Reports will be reviewed and handled confidentially. The maintainer reserves the right to remove comments, edit issues, ban contributors, or take other action consistent with the Contributor Covenant's enforcement guidelines.

For **security vulnerabilities**, please use the private flow in [SECURITY.md](./SECURITY.md) instead. Not a public issue or Discussion thread.
